Il suffit de lancer les Notebooks avec Visual Studio Code. Dans le terminal vous pouvez créer un nouvel environnement avec la commande : 

> python -m venv challenge_jessem 

(challenge_jessem est le nom de l'environnement)

Dans ce cas, un répertoir sera créé dans Challenge.

Ensuite, pour l'installation des bibliothèques, ce sera à faire dans les notebooks, la commande est déjà écrite au début de chaque notebook.
